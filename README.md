# spaceeee

## setup
1. `npm ci`

### set up a server
2. `npm start`
3. copy `cfg.example.json` to `cfg.json` and edit according to your needs.

### set up a client for editing
2. `npm run client-dev`
3. go to http://localhost:3000

### build the client
2. `npm run build-client`
3. find it in `build/`
