# spaceeee

## setup
1. `npm ci`

### set up a server
2. copy `server/cfg.example.json` to `server/cfg.json` and edit according to your needs.
3. `npm start`

### set up a client for developing
2. `npm run dev-client`
3. go to http://localhost:3000

### build the client
2. `npm run build-client`
3. find it in `build/`
